# Student Attendance System 📋 (ASP.NET)

This is a web-based Student Attendance Management System developed using ASP.NET framework.

## 📌 Project Overview
This system is designed to manage student attendance digitally. It includes different master pages for Students, Teachers, and other modules.

## ✨ Features
- Student Management System
- Teacher Management System
- Attendance Marking System
- Reports Generation
- Role-based access (Admin / Teacher / Student)

## 🧩 Master Pages
This project includes 4 master pages:
- Admin Master Page
- Student Master Page
- Teacher Master Page
- Common Master Page 

## 🛠️ Technologies Used
- ASP.NET 
- C#
- HTML, CSS, 
- SQL Server Database

## 🎯 Purpose
This project is developed for managing student attendance efficiently with role-based access for different users.

## 📁 Modules
- Admin Module
- Student Module
- Teacher Module
